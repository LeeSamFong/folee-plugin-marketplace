# Folee Plugin Marketplace

一个轻量的 agent plugin marketplace，用来收录可信的 Agent / Codex 插件入口。

## 使用方式

将本仓库添加为 Codex marketplace：

```bash
codex plugin marketplace add LeeSamFong/folee-plugin-marketplace
```

安装插件：

从下方已收录插件中选择需要的插件，并执行对应安装命令。

## 已收录插件

### Superpowers

- 安装名称：`superpowers@folee-plugin-marketplace`
- 分类：Developer Tools
- 来源：https://github.com/obra/superpowers

```bash
codex plugin add superpowers@folee-plugin-marketplace
```
